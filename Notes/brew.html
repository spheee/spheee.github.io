# mac 环境重装

最近一直在用公司的 ubuntu 做开发，颇为顺手。今天突然把 Mac 掏了出来折腾了几下，竟然发现 brew 、npm 都莫名其妙报错了。可能是升级了 macOS 的原因。只能重装环境

# brew 安装

ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"
1

比较担心的是会不会将之前的安装的包全部删除，也没办法只能硬着头皮来了。
安装的过程比较久，需要一点耐心。可能会下载CommandLineTools
安装好了，来brew doctor 一大堆问题 先清除掉不用的 brew cleanup subl 是sublime 的link 我已经不用了
